# SBTI 人格测试项目

## 项目简介
这是一个幽默风趣的人格测试网站，采用SBTI测试系统。网站以轻松娱乐的方式帮助用户了解自己的性格特征。

## 设计风格
- **主题色**：绿色系(#059669)极简风格
- **风格**：极简主义、简洁、现代
- **交互**：绿色强调、简洁线条、细腻过渡

## 功能特点
- 31道精心设计的测试题目
- 16种人格类型分类
- 幽默诙谐的文案风格
- 响应式设计，适配各种设备
- 实时答题进度跟踪

## 技术栈
- React 18
- TailwindCSS
- Lucide Icons

## 页面结构
- `index.html` - 首页，展示测试介绍
- `test.html` - 测试页面，进行答题
- `app.js` - 首页应用入口
- `test-app.js` - 测试页应用入口

## 组件目录
- `Header.js` - 导航栏组件
- `Hero.js` - 首页英雄区
- `Footer.js` - 页脚组件（含作者信息）
- `TestQuestion.js` - 测试问题组件

## 特色内容
- 作者信息来自B站up主"蛆肉儿串儿"（UID417038183）
- 测试定位：娱乐为主，不作为专业诊断依据
- 友情提示：仅供娱乐，请勿用于盈利
